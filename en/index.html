Multithreaded SSH V0.1.1

WAKE published on 2023-06-24 included in Categories

Introduction

Recently, I replaced a batch of Huawei switches and switched to using SSH for remote access, which rendered my previous Telnet script unusable. Therefore, I spent some time rewriting a multi-threaded SSH program called Multithreaded-SSH using Python.

This version is still not very flexible, but it can execute some basic commands. For example, it can execute “dis cur” command combined with the “sshlog” file to backup device configurations, or perform batch password modifications.
